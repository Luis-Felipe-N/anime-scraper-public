# Anime scraper

Funcionalidades 
- [x] getScraper - Faz scraper de todos os animes raspado do site pegando por gênero.
- [x] getAnimeByGenre - Faz scraper de animes pelo gênero.
- [x] getAnimeBySlug - Faz scraper do anime pelo slug.
